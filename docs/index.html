Run Your Own & Connect

Clone and start the server locally, then point your NTP client at it:

Clone git clone https://github.com/aykutsp/ntp-server.git
Run go run ./cmd/ntp-server -config ./configs/server.example.json
NTP port 127.0.0.1:12300
HTTP API http://127.0.0.1:8080
Client Examples
# Built-in query tool
go run ./cmd/ntp-query -server 127.0.0.1:12300

# Linux — chrony (/etc/chrony.conf)
server 127.0.0.1 port 12300 iburst

# macOS
sntp -S 127.0.0.1

# Windows PowerShell
w32tm /config /manualpeerlist:"127.0.0.1" /syncfromflags:manual /update
w32tm /resync /force

# Python
pip install ntplib
python -c "import ntplib; r=ntplib.NTPClient().request('127.0.0.1',port=12300); print(r.offset, r.stratum)"
